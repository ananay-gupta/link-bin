
# Link-bin
An extension to keep all of your favourite sites


## Run Locally

Clone the project

```bash
  git clone https://github.com/light-200/Link-bin.git
```

Go to the following url

```bash 
  chrome://extensions/
```
Click on the load unpack button and browse to the project location
